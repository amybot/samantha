# Samantha

Distributed bot frontend for Discord.

## Configuration

```Bash
BOT_NAME="unique-name-without-spaces"
BOT_TOKEN="your token here"
CONNECTOR_URL="http://your.shard.connector/"
REDIS_IP="redis-ip"
REDIS_PASS="a"
# V This MUST be the same between frontend and backend!
EVENT_QUEUE="memes"
```
